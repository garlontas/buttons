# SVG Buttons
A collection of embeddable SVG buttons for Markdown Documents!

They are available in following sizes:
- 1024 x 300
- 200 x 59
- 150 x 44

Embed them into your Markdown document like this:

```[![Description here](https://raw.githubusercontent.com/garlontas/buttons/main/buttons/<folder>/<icon name>.svg)```

Replace ```<folder>``` with one of our folders and ```<icon name>``` with the icon you need.
